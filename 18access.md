# Контроль доступа

- Создайте пользователя jhon с паролем «qwerty»
- Создайте роль devs
- Выдайте роли devs права на SELECT на любую таблицу
- Выдайте роль devs пользователю jhon

- Предоставьте результаты SELECT из system-таблиц, соответствующих созданным сущностям


На проверку отправьте текстовый документ, содержащий скрипты для создания пользователя и роли, назначения прав доступа, 
а также результаты запросов из системных таблиц, подтверждающие выполнение этих действий. Рекомендуем размещать работу в отдельном Git-репозитории.
