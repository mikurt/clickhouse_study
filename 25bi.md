## Подключите Superset к базе данных ClickHouse
Установлен драйвер clickhouse-connect

```
sudo docker exec -it --user root 7a1db53e55b7 /bin/bash -c "source /app/.venv/bin/activate && pip install --ignore-installed --force-reinstall --target=/app/.venv/lib/python3.10/site-packages clickhouse-connect"
```
